# Day 2 - Salesforce Platform

## 1. What is Salesforce Platform?

Salesforce Platform is a cloud-based platform used to build CRM applications, automate processes, and manage customer data.

It allows both configuration and coding-based development.

---

## 2. Salesforce Components

### App
A collection of tabs, objects, and features designed for specific business processes.

Example:
Sales App

### Object
Stores data in Salesforce.

Example:
Account, Contact, Opportunity

### Tab
Used to access objects and applications in Salesforce UI.

---

## 3. Configuration vs Coding

| Configuration | Coding |
|---|---|
| No-code/Low-code | Programming required |
| Uses clicks | Uses Apex/LWC |
| Faster development | More customization |
| Admin work | Developer work |
